# Le Trong Hieu — Personal Portfolio

A futuristic, dark-themed personal portfolio website for **Le Trong Hieu**, a first-year Computer Science student at the University of Information Technology (UIT), Ho Chi Minh City, aspiring to become an AI Engineer.

## 🌐 Live Sections

| Section | Description |
|---|---|
| **Hero** | Animated intro with typewriter effect, orbit rings, and particle canvas |
| **About** | Background, core values, and quick stats |
| **Skills** | Categorised skill cards (Languages, AI/ML, Data, Dev Tools, CS Fundamentals) |
| **Projects** | Highlighted personal projects with tech stacks and takeaways |
| **Education** | B.Sc. CS at UIT — coursework, GPA, and highlights |
| **Contact** | Email link and social/resume shortcuts |

## 🛠️ Tech Stack

- **HTML5** — semantic, accessible markup
- **CSS3** — vanilla CSS with custom properties, glassmorphism, animations
- **JavaScript** — typewriter effect, scroll-reveal, custom cursor, particle canvas, mobile menu

## ✨ Design Highlights

- Dark theme with neon cyan / purple / green accents
- Animated orbit rings and aurora gradients on the hero
- Scroll-triggered reveal animations
- Custom cursor with ring-follow effect
- Fully responsive (desktop → tablet → mobile)
- Accessible — ARIA labels, roles, and semantic HTML throughout

## 📁 File Structure

```
Update_portfolio/
├── index.html   # Main markup
├── style.css    # All styles and responsive breakpoints
└── script.js    # Interactivity and animations
```

## 🚀 Running Locally

No build step required — just open `index.html` in any modern browser.

```bash
# Or serve with any static server, e.g.
npx serve .
```
